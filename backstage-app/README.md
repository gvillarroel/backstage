# f(dx) AI Backstage

Backstage application for the f(dx) AI Backstage experience.

To start the app, run:

```sh
corepack yarn install --immutable
corepack yarn start
```

## Route notes

- `/` is the AI Backstage home page.
- `/models`, `/skills`, `/communities`, `/agents`, `/foundations`, `/platform`, and `/showcase` are custom AI product pages.
- `/docs` is the curated AI Backstage documentation hub.
- `/techdocs` is the native Backstage TechDocs index.
- GitHub Pages publishes the curated surface and does not rely on the live TechDocs backend.
