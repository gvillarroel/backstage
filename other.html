EFX color system

Six color palettes work together to create depth and meaning.

The design system uses primary colors for key actions, shadow colors for text on tinted backgrounds, highlight colors for badge backgrounds, status colors for alerts, and a full gray scale from 100 to 900.

All colors are defined as CSS custom properties in :root and referenced via var(--efx-*) throughout.

PlantUML · Activity

Model Evaluation Pipeline

@startuml
!theme plain
skinparam backgroundColor #f7f7f7

start
:Receive model submission;
:Validate metadata;

if (Has required fields?) then (yes)
  :Run automated benchmarks;
  :Measure latency, accuracy, cost;

  if (Passes quality gate?) then (yes)
    :Generate eval report;
    :Submit to Risk Council;

    if (Risk level?) then (low)
      :Auto-approve;
      :Register in catalog;
    else (high)
      :Schedule deep review;
      :Council decision;

      if (Approved?) then (yes)
        :Register with conditions;
      else (no)
        :Return with feedback;
        stop
      endif
    endif
  else (no)
    :Flag issues;
    :Return to submitter;
    stop
  endif
else (no)
  :Reject submission;
  stop
endif

:Notify team;
:Update backstage;
stop
@enduml
